# Student Management System (Java)

A console-based Student Management System built using Java. This project was created to practice Object-Oriented Programming (OOP), Collections Framework, File Handling, Exception Handling, and Git/GitHub workflow.

## Features

* Add new students
* Display all student records
* Search student by ID
* Search student by Name
* Update student details
* Delete student records
* Sort students by ID
* Sort students by Name
* Prevent duplicate Student IDs
* Save data to file automatically
* Load data from file when the program starts
* Handle invalid user inputs using Exception Handling

## Technologies Used

* Java
* ArrayList
* Comparable Interface
* Comparator (Lambda Expressions)
* File Handling (File, FileWriter, Scanner)
* Exception Handling
* Git & GitHub

## How to Run

1. Clone the repository.
2. Open terminal in the project folder.
3. Compile the program:

javac Student.java

4. Run the program:

java Student
